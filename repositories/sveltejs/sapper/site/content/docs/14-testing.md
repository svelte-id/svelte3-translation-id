Kamu dapat menggunakan _framework_ pengujian dan pustaka yang kamu suka. Bawaan (_default_) pada [templat-sapper](https://github.com/sveltejs/sapper-template) adalah [Cypress](https://cypress.io).


### Running the tests

```bash
npm test
```
